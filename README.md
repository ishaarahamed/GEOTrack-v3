# 🌍 GEOTrack v3 — Global Missile Tracking Dashboard

![Status](https://img.shields.io/badge/status-demo-blue)
![Version](https://img.shields.io/badge/version-v3.0-green)
![License](https://img.shields.io/badge/license-MIT-yellow)

GEOTrack v3 is an interactive **global missile tracking and intelligence visualization dashboard** built using **HTML, CSS, and JavaScript**.
It simulates missile launches, conflict zones, and military intelligence on a real-time interactive world map.

⚠️ **Important:** All missile data and conflict scenarios shown in this project are **simulated** and used strictly for **educational and demonstration purposes.**

---

# 🚀 Features

• 🌍 Interactive global map powered by **Leaflet.js**
• 🚀 Simulated missile launch tracking system
• 🛰 **NASA GIBS satellite imagery** integration
• 🗺 Country boundary visualization using GeoJSON
• ⚔ Conflict zone selection panel
• 📰 Global news feed integration (NewsAPI supported)
• 🛰 Sentinel satellite footprint visualization
• 📊 Country-level military intelligence dashboard
• ⚡ Tactical style UI inspired by real command systems

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/geotrack-v3.git
```

Open the project folder and run:

```
GEOTrack-v3-final.html
```

in any modern browser.

No build tools or frameworks required.

---

# 🔑 API Setup (Optional)

To enable live news and satellite integrations, update the configuration inside the HTML file:

```javascript
const CFG = {
  NEWS_API_KEY: '',
  COPERNICUS_USER: '',
  COPERNICUS_PASS: ''
};
```

You can register APIs from:

• https://newsapi.org
• https://scihub.copernicus.eu
